# Personalized Networking Assistant

## Project Description
This project is an AI-powered networking assistant that helps users generate conversation starters, analyze event themes, and perform fact-checking using Wikipedia.

## Technologies Used
- Python
- FastAPI
- Streamlit
- Transformers
- GPT-2
- Wikipedia API

## Features
- Event Theme Detection
- Conversation Starter Generation
- Fact Checking
- History Logging
- Feedback Logging

## Run the Project

Start FastAPI:

uvicorn main:app --reload

Start Streamlit:

streamlit run frontend/app.py
